1.Food Ordering and Management System

This was our final year project for my bachelors degree. The project is a web based software which focuses on our college canteen.The user can order food items from the web based application after creating an account.If the useer already have an account he/she can either use phone number and password or can use the OTP facility in order to login to the account. OTP can be recieved either by a call or by a text message which the user can select according to their convenience .After user/customer places an order the admin side will recieve a notification and the canteen manager can decide weather or not to take the order.After the status update of the order the user will recieve the status as a text message with the waiting time. we have also integrated payment gate way using razor pay inoredr to promote cashless transations
